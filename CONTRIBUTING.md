# Contributing

This repository documents the public NeuroCAD integration surface.

## Contribute changes that improve

- contract clarity
- schema consistency
- example quality
- public integration guidance

## Before opening a pull request

- keep public names and paths stable
- update examples when the contract changes
- update `CHANGELOG.md` for externally visible changes
- run the validation workflow locally when possible

## Out of scope

Do not add private kernel internals, deployment secrets, or internal-only orchestration details.
