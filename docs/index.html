Getting Started

Minimum public integration loop

  1. Exchange client credentials for a bearer token.
  2. Create or resolve a project container.
  3. Submit an inference or export job.
  4. Poll job state or receive a webhook.
  5. Resolve artifact metadata and download references.

Environments

Sandbox-first operating model

  • Production: https://api.neurocad.eu
  • Sandbox: https://sandbox.api.neurocad.eu
  • Separate credentials and callback endpoints per environment
  • Webhook consumers should treat payloads as untrusted input
  • Clients should implement backoff behavior for 429 responses

Example

Submit a job

curl -X POST https://sandbox.api.neurocad.eu/v1/inference/jobs \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "project_id": "prj_demo_001",
    "task": "mesh-generation",
    "input": {
      "material": "steel-a36",
      "resolution": "medium"
    }
  }'
